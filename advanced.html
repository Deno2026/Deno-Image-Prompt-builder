연결 방식까지 포함해서
조금 더 자유롭게 다루는 페이지

이 페이지는 입문자용과 다르게, 직접 작성하는 방식부터 로컬 LLM과 API LLM을 붙이는 흐름까지 함께 다룰 수 있도록 설계했습니다. 필요한 방식만 열어 쓰도록 나눠두었기 때문에, 기능은 많아도 한 번에 전부 보이지 않게 정리했습니다.

지금 버전은 고급 사용자를 위한 베타 공개본입니다. 연결용 입력칸과 결과 패널을 분리해두어 이후 실제 호출까지 붙이기 쉽게 해두었습니다.

워크플로우 선택

고급모드에서도 모든 기능을 한 번에 보지 않도록, 먼저 작업 방식을 고르고 그 안에서 필요한 항목만 다루게 구성했습니다.

실시간 초안

현재 입력을 바탕으로 바로 복사할 수 있는 고급 프롬프트 초안을 만듭니다.

호출 준비 페이로드

로컬 LLM 또는 API LLM에 넘기기 전에 어떤 구조로 보내면 좋을지 미리 확인할 수 있습니다.

권장 사용 흐름

  • 처음엔 `직접 작성`만 채워도 됩니다.
  • 로컬이나 API 연결은 자신이 실제로 쓰는 방식만 열어두면 됩니다.
  • 후처리는 결과가 너무 길거나 과장될 때 정리용으로 붙이면 좋습니다.

현재 모드 요약

활성 모드직접 작성 결과 성격이미지 프롬프트 연결 방향수동 작성 중심
